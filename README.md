1. Plik pp5-key.json trzeba skopiować do folderu głównego projektu
   
2. Należy zainstalować pakiety:
npm install googleapis
npm install express googleapis body-parser
npm install cors

oraz odpalić serwer:
node server.js (otwiera się na porcie 2137)
